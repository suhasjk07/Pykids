Frequently Asked Questions
Is Pykids really free? ▾
Yes — the full curriculum from Foundations through Advanced Python is completely free. A future optional premium tier may offer certificates and additional projects, but the core learning experience will always be free.
What age is Pykids designed for? ▾
The curriculum is designed to be accessible from age 11 (Grade 6 / Year 7) upward. Younger gifted students may also find it accessible. There is no upper age limit — many adults use Pykids to learn Python from scratch too.
Does my child need to install anything? ▾
No. Pykids runs entirely in the web browser. Students can write and run Python code directly on the site. The full Python runtime (Pyodide) runs in WebAssembly in the browser — nothing to download or install.
How long does the full curriculum take? ▾
At a comfortable pace of 30–45 minutes per day, most students complete Foundations in 2–3 weeks, and the full curriculum including Full-Stack Python in 6–12 months. The platform is self-paced — there's no pressure to rush.
Can teachers use Pykids in a classroom? ▾
Absolutely. Pykids provides a Teacher Dashboard where educators can create classes, add students, assign lessons, and track progress. All materials are available for classroom use at no cost. Contact us for bulk school licenses if needed.
What technology stack does Pykids use? ▾
The frontend is built with Next.js and Tailwind CSS. The backend uses Python with FastAPI. The in-browser Python runtime uses Pyodide (WebAssembly). User auth and data are managed via Supabase. Everything is hosted on Vercel (frontend) and Render (backend). All tools used have generous free tiers.
